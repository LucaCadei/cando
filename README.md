# Cando — Marketplace di concetti

## Avvio

**Backend**
```bash
cd backend
uv run uvicorn main:app --reload
# API disponibile su http://localhost:8000
# Docs interattive: http://localhost:8000/docs
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
# App disponibile su http://localhost:5173
```
